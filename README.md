# Smart Contract Notes Repo

Beginner-friendly Solidity notes with examples, wrapped in a polished local website with motion, search, progress tracking, a quiz, a glossary, and copy-ready contracts.

## Open the Website

Option 1: double-click `index.html`.

Option 2: run the local server:

```powershell
npm start
```

Then open:

```text
http://localhost:4179
```

On Windows, `start.bat` also starts the local server and uses `pushd` so it works better from a UNC/Desktop folder.

## What Is Included

- `index.html` - the static website shell.
- `styles.css` - responsive UI, themes, motion, and animations.
- `app.js` - lessons, search, progress, quiz, glossary, copy actions, and canvas animation.
- `assets/solidity-contract-lab.png` - project-local hero visual.
- `notes/` - Markdown notes for the learning path.
- `contracts/` - beginner Solidity examples.
- `server.mjs` - tiny dependency-free local static server.
- `start.bat` - Windows launcher.

## Learning Flow

1. Start with Contract Anatomy.
2. Read each compact note.
3. Copy the linked contract into Remix, Hardhat, or Foundry.
4. Mark the lesson complete.
5. Use the quiz and glossary for quick review.

These examples are educational and are not audited production contracts.
